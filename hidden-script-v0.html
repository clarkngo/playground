Exercise: The Hidden Script

In this activity, you will prevent Cross-Site Scripting (XSS) by sanitizing user input.

Scenario: A user posted this comment: <script>alert('HACKED')</script>. Because your code blindly returns it, the browser thinks it is real code and executes it!

Your Task

Modify the render_comment function to make the input safe:

  1. Import HTML: Import the python library html.
  2. Escape the Input: Use html.escape(comment) to convert special characters into safe text (e.g., < becomes &lt;).
  3. Return Safe Text: Return the escaped string.

Why This Matters

Escaping ensures the browser treats data as content, not executable code. This is the primary defense against XSS attacks.
