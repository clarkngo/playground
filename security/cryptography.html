Exercise: Secure User Registration

In this activity, you will implement a Salted Hash function to store user credentials securely.

Scenario: You are building the registration system for a new app. You must ensure that every user has a unique salt generated for them, so that identical passwords result in different hashes.

Your Task

Complete the save_user function:

  1. Generate Salt: Use secrets.token_hex(16) to create a random string.
  2. Create Hash: Combine the salt and password, then hash them using hashlib.sha256.
  3. Return Data: Return a dictionary containing both the salt and the final hash.

Why Return Both?

When the user tries to login later, you will need that specific salt to verify their password. We store both in the database.
